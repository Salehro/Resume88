Hard Skills

Teachable, measurable technical skills like languages or software.

Hard skills are specific, teachable, and measurable technical abilities — things like programming languages, software tools, methodologies, languages spoken, and certifications. They're contrasted with "soft skills" (interpersonal qualities like communication and teamwork) and are usually the skills a job description explicitly lists as required.

Examples of hard skills include: Python, SQL, Figma, Salesforce, Excel, Google Analytics, Adobe Photoshop, Spanish (C1 level), AWS certifications, financial modeling, and SEO. On a resume, hard skills belong in a dedicated Skills section and should also appear in your bullet points showing how you applied them.

Example: A software engineer's hard skills: "Python, TypeScript, React, PostgreSQL, AWS, Docker, GraphQL." A marketer's: "Google Analytics, HubSpot, SEMrush, Meta Ads, Figma, SQL."
